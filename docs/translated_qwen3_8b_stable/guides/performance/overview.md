title: 性能指南
---

DuckDB 旨在通过选择良好的默认配置和具有容错性的架构自动实现高性能。当然，仍然有机会针对特定工作负载对系统进行调优。性能指南页面包含有关在使用 DuckDB 加载和处理数据时实现良好性能的指南和技巧。

指南中包含几个微基准测试。您可以在 [基准测试页面]({% link docs/stable/guides/performance/benchmarks.md %}) 找到这些详细信息。
